# Система распознавания объектов на видеопотоке в реальном времени (YOLO)

Программная система для анализа видеопотока в реальном времени и распознавания
объектов заданных классов: людей, автомобилей, спецтехники, а также средств
индивидуальной защиты (каски, жилеты). Архитектура строится на предобученных
моделях семейства YOLO и остаётся универсальной: замена модели или дообучение
на новом датасете позволяет адаптировать систему под любую предметную область —
от контроля пропускного режима до мониторинга техники безопасности.

Ключевой конвейер обработки (захват видео → инференс модели → постобработка →
визуализация) не зависит от набора классов, что позволяет последовательно
наращивать функциональность, начиная с базовой детекции объектов COCO и
добавляя специализированные модули (СИЗ, номерные знаки, промышленные объекты).

## Основные возможности

- **Базовая детекция объектов.** Распознавание людей и транспорта «из коробки»
  на предобученной модели YOLOv8/yolov8s (датасет COCO, 80 классов).
- **Фильтрация по классам и порогам уверенности.** Поддержка целевых классов
  (`person`, `car`, `truck`, `bus`), настройка `conf` и `iou` под конкретный
  сценарий.
- **Дообучение на кастомных классах.** Адаптация модели под средства
  индивидуальной защиты (каска, жилет) и спецтехнику через fine-tuning на
  датасетах Roboflow Universe.
- **Распознавание номерных знаков.** Локализация автомобиля → вырезка области
  номера → OCR (EasyOCR) → нормализация текста → проверка по белому списку.
- **Логика принятия решений.** Белый список номеров (допуск/отказ), детекция
  нарушений СИЗ, формирование алертов с сохранением кадра-доказательства.
- **Оптимизация производительности.** Half-precision (FP16), экспорт в ONNX и
  TensorRT, пропуск кадров — для достижения целевых 10–15 FPS.

## Технологии

- **Python 3.9–3.11** — основной язык реализации.
- **PyTorch + CUDA** — бэкенд для инференса и дообучения (CPU-версия как fallback).
- **Ultralytics** — высокоуровневый API к моделям YOLOv8 (тренировка, инференс, экспорт).
- **OpenCV** — захват видеопотока, отрисовка bounding boxes, предобработка кадров.
- **EasyOCR** — распознавание текста на номерных знаках (кириллица + латиница).
- **ONNX Runtime / TensorRT** — ускоренный инференс оптимизированных моделей.

## Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/<твой-username>/<твой-репозиторий>.git
cd <твой-репозиторий>

# 2. Создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate        # Linux/macOS
# .venv\Scripts\activate         # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. (Опционально) Проверить доступность GPU
python -c "import torch; print('CUDA:', torch.cuda.is_available())"

# 5. Запустить базовую детекцию с веб-камеры
yolo predict model=yolov8s.pt source=0
```

## Использование

```bash
# Базовый конвейер: видео → детекция → отрисовка
python src/pipeline.py --source 0 --model yolov8s.pt --conf 0.25

# С фильтрацией по классам (только люди и автомобили)
python src/pipeline.py --source data/test.mp4 --classes person car truck bus

# С дообученной моделью и логированием событий
python src/pipeline.py --source 0 --model runs/detect/train/weights/best.pt --log events.csv
```
